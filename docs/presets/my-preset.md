# My Preset Preset

This preset is inspired by [Tokyo Night](./tokyo-night.md) preset.

![Screenshot of My Preset preset](/presets/img/my-preset.png)

### Prerequisites

- A [Nerd Font](https://www.nerdfonts.com/) installed and enabled in your terminal

### Configuration

```sh
starship preset my-preset -o ~/.config/starship.toml
```
